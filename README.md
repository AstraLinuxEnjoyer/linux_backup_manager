# Linux Backup Manager

Графический менеджер резервного копирования для Astra Linux 1.8.

Основной движок резервного копирования: `borgbackup`.

## Возможности

- резервное копирование выбранных каталогов;
- инкрементальные копии с дедупликацией;
- максимальное сжатие через `lzma,9`;
- восстановление выбранного архива одной кнопкой;
- проверка целостности репозитория и архивов;
- журнал выполнения команд в интерфейсе;
- запуск из директории проекта через исполняемый файл `linux-backup-manager`.

## Зависимости

Для Astra Linux 1.8 установите:

```bash
sudo apt update
sudo apt install borgbackup python3-tk
```

## Запуск

```bash
chmod +x linux-backup-manager
./linux-backup-manager
```

## Важные замечания

- Для полной копии системы запускайте программу с правами администратора:

```bash
sudo ./linux-backup-manager
```

- Не сохраняйте репозиторий резервных копий внутри каталогов, которые копируете.
- Для полного восстановления системы рекомендуется загрузиться с LiveUSB/установочного носителя, смонтировать целевой раздел и восстановить архив в точку монтирования.
- При копировании `/` программа автоматически исключает виртуальные и временные каталоги: `/proc`, `/sys`, `/dev`, `/run`, `/tmp`, `/mnt`, `/media`, `/lost+found`.
